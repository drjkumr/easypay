# Easypay – The Payroll Management System

**Author:** Dheeraj  
**Organization:** Hexaware Technologies Limited  
**Tech Stack:** Java, Spring Boot, React.js, MySQL, JWT, Docker, AWS/Azure

---

## 🚀 Project Overview

Easypay is a robust and scalable Payroll Management System designed for XYZ Company to streamline payroll operations, ensure compliance, and enhance employee satisfaction. It automates salary calculations, manages leave and benefits, and provides secure access to payroll data for all stakeholders.

This project is built with a strong focus on **auditability**, **modularity**, and **real-world business logic**, aiming to deliver enterprise-grade impact.

---

## 🎯 Objectives

- ✅ Automate payroll calculations with precision and auditability  
- ✅ Ensure compliance with tax and labor regulations  
- ✅ Provide a secure self-service portal for employees  
- ✅ Enable timely and accurate salary disbursement  
- ✅ Support scalability and maintainability for future growth  
- ✅ Integrate analytics and GenAI tooling for enhanced admin experience  

---

## 🧩 System Architecture

### 🔐 Authentication & Authorization
- JWT-based login and role-based access control
- Secure token transmission and validation
- Logout flow with token invalidation

### 🗃️ Backend (Spring Boot)
- RESTful APIs with Swagger documentation
- ORM integration with MySQL
- API versioning and property-based configuration
- Unit testing with mocked dependencies
- Audit trail and changelog tables for compliance
- Exception handling with user-friendly messages
- Modular service layer for salary, bonus, and leave logic
- Integration with Python microservices for analytics

### 🎨 Frontend (React.js)
- Responsive UI with Bootstrap/Flexbox
- Form validation and error handling
- Routing and navigation
- Secure login and registration pages
- Employee dashboard and leave request module
- Admin dashboard with GenAI assistant for queries and insights

## 👥 User Roles & Use Cases

| Role               | Key Use Cases |
|--------------------|---------------|
| **Admin/HR Manager** | Manage employees, define payroll policies, generate reports, view audit logs |
| **Payroll Processor** | Calculate and verify payroll, manage benefits, process payments |
| **Employee**         | View pay stubs, update info, submit timesheets, request leave |
| **Manager/Supervisor** | Review team payrolls, approve leave requests |
